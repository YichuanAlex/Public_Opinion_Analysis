怎么获取 Cookie：
1. 在 Chrome 打开 xiaohongshu.com 并登录
2. 按 F12 打开开发者工具 → 点「Network（网络）」标签
3. 刷新页面，点左侧列表里任意一条 www.xiaohongshu.com 请求
4. 右侧「Headers（标头）」→ 找到「Cookie」→ 右键复制整个值
5. 粘贴到下方，点「保存」

模型会学习什么

标题、正文、封面图、当前互动量和互动结构，以及这些信号在历史投放里的 CPE 和总互动表现。

默认成功定义

综合 CPE 越低越好，同时参考总互动规模。你可以在左侧切换为手动 CPE 阈值。

适用边界

历史样本越多判断越准，适合做投前初筛和小额测试建议，不适合当作绝对结论。工具会显示置信度和相似样本。
